Installation of the Console Application

The following is just a recommendation on using the output, describing how I do it.

  1. Load the output file into Excel or any other spread-sheet processing program.
  2. Sort the output using two filters. First, sort by 'type' to get all results of type "combined" to the top and then sort by p.value_PSI.

You could also sort by the coverage ratio p-value, but personally I think the PSI score p-value works better. You could still apply a filter on the coverage ratio p-value column to reduce the number of results.

If you are still unsatisfied with the length of the result list, you can filter for events that use known junctions only and/or apply a filter to the absolute coverage ratio change and PSI effect size. Larger changes might indicate a more important event. Usually I don't apply these further filtering steps for exon skipping events.

  1. Remove duplicated entries based on the gene symbol. This way you will get a 'ranked' list of genes that might have alternative splicing events and you don't have to keep in mind which genes you looked at already.
  2. Enter the gene names in the Manananggal web interface and verify the finding.
